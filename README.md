# Gesture-Controlled Smart Mirror

This project is a 10"x12" smart mirror with a display behind a two-way acrylic mirror,
controlled by a Raspberry Pi and a gesture sensor (touch-free control).

## Goals
- Build a clean mirror enclosure (minimal visible wiring)
- Run a smart mirror UI (MagicMirrorÂ² or similar)
- Add gesture controls for basic navigation

## Planned Parts
- Two-way acrylic mirror (10"x12")
- 7" HDMI display
- Raspberry Pi 4
- APDS-9960 gesture sensor
- Wooden frame + plywood back panel
